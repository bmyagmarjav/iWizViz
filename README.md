## Prerequisites

* [node.js](https://nodejs.org/) - Spectacular Test Runner for JavaScript

```sh
$ sudo npm install -g yo
$ sudo npm install -g generator-fountain-webapp
$ sudo npm install -g gulp-cli
```

## Installation

```sh
$ cd iWizViz
$ sudo npm install
```
