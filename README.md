## Prerequisites

* [node.js](https://nodejs.org/)

```sh
$ sudo npm install -g yo
$ sudo npm install -g bower
$ sudo npm install -g generator-fountain-webapp
$ sudo npm install -g gulp-cli
```

## Installation

```sh
$ cd iWizViz
$ sudo npm install
$ sudo bower install
```

## Build, Run, Test

```sh
$ gulp build
$ gulp serve
$ gulp test
```
